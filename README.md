# Nafiz Ahmed - Portfolio Website

A modern, creative portfolio website showcasing my work as a Machine Learning Engineer and Researcher.

## Features

- Dark theme with vibrant gradient accents
- Smooth scroll animations
- Typewriter effect for role cycling
- Live iframe previews for project demos
- Fully responsive design
- Interactive timeline for experience
- Parallax effects

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Space Grotesk, Inter)

## Sections

1. **Hero** - Introduction with animated profile and stats
2. **About** - Bio and tech stack
3. **Publications** - Research papers and ongoing work
4. **Projects** - Featured projects with live demos
5. **Experience** - Professional timeline
6. **Contact** - Social links

## Local Development

Simply open `index.html` in your browser.

## Deployment

Push to GitHub and enable GitHub Pages in repository settings.
